node nvm 以及 angular cli install教學

Created2023-01-21|Updated2023-01-21
|Post View:

我們採取使用 nvm 管理我們的 Node.js
參考網址: https://www.casper.tw/development/2022/01/10/install-nvm/
常用命令:
nvm ls-remote：列出目前可用的遠端 Node.js 版本
nvm install ：安裝特定版本的 Node.js
nvm ls：列出本定端所安裝的 Node.js 環境
nvm alias default node：設定命令列預設開啟的 Node.js 版本
nvm use：當前命令列套用特定版本的 Node.js

Angular CLI

npm i -g @angular/cli

mac angular cli 必須要加上 sudo 給予安裝權限 才成功安裝

安裝好 cli 要先輸入此 而不是 ng –version
ng version 現在新版本會跳出 問你幫不幫助他們收集資料，最後就安裝好
