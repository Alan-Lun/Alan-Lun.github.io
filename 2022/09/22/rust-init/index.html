Rust初學第一步簡單介紹

Created2022-09-22|Updated2022-09-22
|Post View:

Rust 官方文章介紹 Rust 應用到 Web 上的現況，摘要：
Rust 已經可以應用到 Web 上了，而且超快！
Production 級的 frameworks：Actix Web、Rocket
新的 frameworks：Wrap、Tide
ORM 有 Diesel
SQL 有 sqlx
MongodDB（官方的）、SQlite、Postgres、MySQL 都有支援
logging、auth、templating、email 都沒問題
還沒有 Rails、Django、Laravel 這種等級的 framework，目前要自己東拼西湊，比較接近 flask 或 express.js
Rust 可以 compile 成 WebAssembly 再跑在 Web 上，例如：Yew、Seed
上手建議在吸收完 The Rust Programming Language 後，再看文中 Getting started 的部分
